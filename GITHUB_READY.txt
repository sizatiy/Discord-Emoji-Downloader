## To Upload to GitHub

1. Create a new repository on GitHub
2. `git init` in this folder
3. `git add .`
4. `git commit -m "Initial commit"`
5. Add remote: `git remote add origin https://github.com/YOUR_USERNAME/discord-emoji-downloader.git`
6. `git push -u origin main`

## For Releases

1. Build the .EXE: Run `build_exe.bat`
2. Find it in `dist/emoji_downloader.exe`
3. Create a GitHub Release and upload the .EXE
4. Users can then download and use it directly!
